MathOp

MathOp enum

Enumeration of mathematical operators supported in Balbismo expressions.

These operators represent the basic arithmetic operations that can be performed on integer and floating-point values. The operators are mapped from their string representations in the source code to enum values for internal processing.

Example Balbismo code:

int result = a + b * c - d / e % f;  // Uses add, mul, sub, div, mod
