SymbolTable createChild()

Creates a new child symbol table for nested scope management.

This method implements lexical scoping by creating a new symbol table that has the current table as its parent. Variables declared in the child scope can access variables from parent scopes, but not vice versa.

Returns: A new SymbolTable instance with this table as its parent.

Example:

final globalScope = SymbolTable();
final functionScope = globalScope.createChild();
final blockScope = functionScope.createChild();

Implementation

SymbolTable createChild() {
  final child = SymbolTable();
  child._parent = this;
  return child;
}
